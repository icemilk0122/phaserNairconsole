# Setting

generator-phaser

  1. Install Node.js
  2. npm install -g yo generator-phaser
  3. mkdir gamefolder
  4. yo phaser

  5. npm start
  6. npm run build
# State

  1. The Boot State: Start the physics system
  2. The Preloader State: Load assets
  3. The Menu State: Game menu
  4. The Game State: Game logic
  5. The Gameover State: Game over and go back to menu
